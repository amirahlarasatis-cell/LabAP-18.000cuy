Deskripsi program:
Program ini merupakan aplikasi manajemen parkir mobil dan inventaris ritel berbasis C++ yang dibuat untuk membantu pengelolaan data secara lebih terstruktur dan efisien. Pada sistem parkir, program menyediakan fitur pencatatan kendaraan masuk, penampilan daftar kendaraan yang sedang parkir, perhitungan biaya parkir berdasarkan lama parkir, proses pembayaran, serta pengelolaan kapasitas parkir maksimum. Sementara itu, pada sistem inventaris ritel, program menyediakan fitur penambahan data barang, penampilan seluruh data inventaris, serta pencarian barang dengan harga tertinggi. Data kendaraan dan barang disimpan menggunakan struct dan dikelola dengan vector sehingga dapat menampung data secara dinamis. Selain itu, program dilengkapi dengan validasi input untuk memastikan data yang dimasukkan pengguna tetap valid, seperti mencegah harga dan stok bernilai negatif serta menghindari kesalahan input. Perhitungan waktu parkir dilakukan menggunakan library <ctime>, sedangkan pengelolaan data memanfaatkan berbagai konsep dasar pemrograman C++ seperti fungsi, percabangan, perulangan, struktur data, dan validasi input. Dengan fitur-fitur tersebut, program dapat membantu proses pengelolaan parkir maupun inventaris secara lebih mudah, cepat, dan efektif.
